Welcome!

Math Lobby is an open source software that provides different tools designed to make
mathematics easier: a calculator, a prime number finder and a binary converter.
New features will be added in the future: if you have any idea feel free to contact me.
Here are some example of the tools: the green text is the user's input.






The Calculator is easy to use and works with both integer and decimal numbers. You can
select which feature you want to use: currently there are Sum, Subtraction, Multiplication,
Division, Sine, Cosine and Tangent, with many more to come.


1.0.1 Patch Notes regarding Calculator:

Square root added to Calculator - Fixed Calculator bugs and improved optimisation.


The Prime Calculator allows the user to find out whether any integer number is prime or not.
It is extremely quick, even with very big numbers, and easy to use. If you want to see
which numbers are prime in a certain range, then the tool you are looking for is Prime List (see
below).
